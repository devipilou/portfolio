Mon Portfolio

Projet Bootstrap et Js

Mon portfolio est l'occasion pour moi de mettre en pratique mes cours sur bootstrap et de réaliser une animation en Javascript.

Il est construit à l'aide de bootstrap5 et du thème Spacelab de Bootswatch et un peu de custon Css.

Le fond de page est une animation Js a l'aide de la librtairie tsParticles.

C'est le site sur lequel vous naviguez actuellement.

Blog de MaitresseGe

Projet Wordpress

Ce projet est pour moi presque un prétexte pour découvrir Wordpress. Qu'on apprécie ou non ce CMS, il n'en est pas moins un incontournable outil web et il est essentiel d'en connaitre l'utilisation.

Ce site est construit sur le thème OceanWP, et à l'aide d'extensions comme Elementor, Wordfence, ou ContactForm7. Il fait également appel à une api smtp gmail pour gérer les envois de mail depuis le formulaire.

C'est le blog dans lequel l'instagrameuse @maitresse.ge partage ses ateliers avec d'autres enseignants de cycle 2.

Galerie de Pilou_art

Projet "from scratch" avec Scss et Js

Ce site est une galerie des dessins que je fais à mes heures perdues et que je partage habituellement sur instagram

Je l'ai concu comme un exercice de construction de site en partant de rien, et en gérant le style en Sass/scss. Les animations du menu sont en Javascript.

Il est installé en tant que sous-domaine du site actuel, et est géré comme sous-module de ce dernier avec Git
